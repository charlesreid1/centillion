# Notes on Disqus API

Start with the [using the api](https://help.disqus.com/developer/using-the-api)
instructions at the disqus help site.

First step is to [create an api application](https://help.disqus.com/api/how-to-create-an-api-application).

Treat this like the Google Drive OAuth app:
have a standalone utility that can be
run once to authorize with a DCPPC Disqus
account, then store the user credentials,
instead of the OAuth app credentials.  

From the private-www mkdocs.yml we know the name
of the comments forum that is embedded on the site:

dcppc-internal

documentation for [all endpoints](https://disqus.com/api/docs/)

below is simple example of how we can call all threads. still have to get next.
